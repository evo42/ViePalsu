How to use Palsu

Palsu is an interactive meeting tool that is used to demonstrate capabilities of VIE. The idea is to create a tool for coordinating both online and real-world meetings.

For further information see also the IKS Project Wiki page

Report issues and feature requests at our issue tracker.

Adding new meetings and real-time sync

Source: palsu.me - adding a new meeting / message sync [demo 1]

In this screencast you'll see how to add a new meeting to Palsu and how the real-time sync via VIE works.

Changes to the meeting title, agenda or messages are synced in real-time across the logged-in clients/participants. Double-click to edit the meeting title and agenda. Single-click to add a chat message.
The left client is using Safari, in the middle there is one participant using Firefox and the right one is using Google Chrome.

All active/online participants in a meeting are indicated with a green square in the "Participants" sidebar on the right side.

Extract entities with Apache Stanbol and add manual annotations

Source: palsu.me - extract entities, add manual annotations [demo 2]

All chat messages are analyzed with Apache Stanbol -- found entities are added to the "Mentions" sidebar.

In this demo your Twitter contacts are added to your "contact repository". You can use them for mentions in chat messages -- select the name of a person you want to annotate, click the person button in the editor and select who you want to mention. Type some chars of the persons name or just use the arrow up/down buttons.
Note that only the last 100 contacts are displayed and the selection currently only works with "enter" (there's a bug when using a mouse-click).

To add a task to the meeting just select the subject of the task, click the clock symbol select a person like before and then the due to date.

In the task view tasks can be filtered by ownership or status. Click on a task in the sidebar or task list to change the complete status -- this is also synced in real-time accross the active clients.

Extra features

Source: palsu.me - embed media/data via oEmbed [demo 3]

If you want to embed media like images, slides, videos, wikipedia articles or tweets just paste the URL into the chat input field -- currently we're using oohembed.com as oEmbed service.
